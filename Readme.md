# To-Do 

This project is a simple To-Do List application . It uses MongoDB, Express, React, and Node.js.

## Project Structure
- **/frontend**: Contains the React.
- **/backend**: Contains the Node.js server and routes.

## How to Setup
1. Open the `backend` folder and run `npm install`.
2. Create a `.env` file based on `.env.example`.
3. Start the server using `node server.js`.
4. Open the `frontend` folder, run `npm install`, and start using `npm run dev`.

## API Endpoints
- `GET /api/todos`: Fetches all tasks.
- `POST /api/todos`: Saves a new task.
